Projects

I am a big fan of personal projects. Since 2009, I've made 8 web apps. I built them to scratch my own itch. Some got accidentally popular because other people found them useful too.

I will admit that some of these apps are old, don't work well (by today's standard), or might not be functional at all (due to API changes). Treat it more like an archive than a list of you should use today.

  • Coursica lets you browse Harvard's course catalog and plan your classes. Over 40,000 courses have been viewed on the app. There are native iOS, Android, and Meteor-based web versions.
  • BTCBot is an automated Bitcoin trading bot. It served me well and has now been retired. (Sensitive source code was removed.)
  • Run Tracker lets you track your GPS location while you move on your phone. I wanted to experiment with React-Native and built this app.
  • Deep Crimson is an AI chessplayer robot built for CS182. It achieved an Elo rating over 1600, meaning it play competitively in tournaments.
  • Microsoft Meet fosters connections between Microsoft employees offering a chance to connect with others. Built for the One Microsoft Hackathon.
  • Julian Date Converter is an Android app for converting between Gregorian datetimes and Julian datetimes. I put it on the app store back in 2012, and it has been downloaded by 1000s.
